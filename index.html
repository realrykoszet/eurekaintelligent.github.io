HOW IT WORKS

8 numbers. Not the document.

01 / FINGERPRINT

8-number summary

TokenSave computes a fingerprint of each document locally — 8 numbers in ~0.1ms. Your document text never leaves your machine.

02 / COMPARE

Compare to memory

The fingerprint is compared to everything your agent has already seen. If it's too similar — it's skipped. If it's new — it passes.

03 / DECIDE

Load only what's different

Documents tagged urgent or critical always pass. Everything else — only if it's genuinely new information.
